Pseudo-elementos

Pseudo-elementos são usados para adicionar estilos a partes específicas de um elemento.

/* Exemplo de pseudo-elemento */
p::before {
    content: "Texto antes!";
    color: red;
}

Pseudo-classes

Pseudo-classes são usadas para definir o estilo de um elemento em um estado específico.

/* Exemplo de pseudo-classe */
a:hover {
    color: green;
}

Combinadores de Seletores

Combinadores permitem aplicar estilos a elementos baseados em sua relação com outros elementos.

/* Exemplo de combinadores */

    • Cores

    Existem várias formas de definir cores no CSS, como nomes, hexadecimal, rgb, e hsl.

    /* Exemplos de cores */
p {
    color: red; /* Cor por nome */
    background-color: #f00; /* Cor hexadecimal */
    border: 2px solid rgb(255, 0, 0); /* Cor RGB */
}

    Propriedades de Texto e Fonte

    Você pode controlar a aparência do texto com várias propriedades, como cor, fonte, tamanho, e alinhamento.

    /* Exemplos de propriedades de texto */
p {
    font-family: Arial, sans-serif;
    font-size: 16px;
    text-align: center;
    color: #333;
}

    Propriedades que Alteram o Tamanho do Elemento

    Você pode controlar o tamanho de um elemento usando propriedades como largura (width), altura (height), padding e margin.

    /* Exemplos de propriedades de tamanho */
div {
    width: 300px;
    height: 200px;
    padding: 20px;
    margin: 10px;
}

    Outras Propriedades

    CSS tem uma vasta gama de propriedades que permitem personalizar ainda mais os elementos, como border-radius, box-shadow, etc.

    /* Exemplos de outras propriedades */
div {
    border-radius: 10px;
    box-shadow: 2px 2px 5px rgba(0, 0, 0, 0.1);
}

    Reset CSS

    O reset CSS é usado para garantir que os estilos padrão dos navegadores não interfiram no design da página.

    /* Exemplo de reset CSS */
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

    Variáveis CSS

    As variáveis CSS permitem armazenar valores que podem ser reutilizados ao longo do código.

    /* Exemplo de variáveis CSS */
:root {
    --cor-principal: #3498db;
}

div {
    background-color: var(--cor-principal);
}
