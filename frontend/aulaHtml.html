Exercícios

  1. Crie uma página com um cabeçalho, dois parágrafos e uma lista.
  2. Adicione um link absoluto e outro relativo na sua página.
  3. Use tags semânticas para organizar um artigo e uma seção.
  4. Inclua uma imagem com caminho relativo e uma linha horizontal.
  5. Organize seu código com indentação adequada.

Mini Projeto

Crie uma página pessoal com:

  • Um cabeçalho com título e navegação.
  • Seção "Sobre Mim" com um parágrafo e uma imagem.
  • Seção de hobbies com uma lista não ordenada.
  • Formulário de contato com campos para nome, e-mail e mensagem.
  • Rodapé com links para redes sociais.
