O que é CSS?

CSS significa Folhas de Estilo em Cascata e ajuda a separar a estrutura do HTML do design e da apresentação visual.

Estrutura do CSS

Exemplo básico de regra CSS:

/* Seletor */
p {
    /* Declarações */
    color: blue;
    font-size: 16px;
}

Seletores

  • Universal: Aplica a todos os elementos. Exemplo: * { margin: 0; }
  • Tag: Aplica a uma tag específica. Exemplo: p { color: red; }
  • Classe: Identificada com ".", aplicada a múltiplos elementos. Exemplo: .classe { font-weight: bold; }
  • ID: Identificada com "#", aplicada a um único elemento. Exemplo: #id { text-align: center; }

Efeito Cascata

Regras definidas posteriormente no código sobrescrevem regras anteriores para o mesmo elemento.

Este texto será azul.

Este texto será vermelho, pois foi definido depois.

Maneiras de Escrever CSS

  • Inline: Estilo diretamente no elemento. Exemplo: <p style="color: red;">Texto</p>
  • Interno: Dentro de uma tag <style> no cabeçalho do HTML.
  • Externo: Em um arquivo .css vinculado. Exemplo: <link rel="stylesheet" href="estilo.css">
